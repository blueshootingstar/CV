# 实验4：PCA + KNN 人脸识别参数说明

## 1. 实验数据与任务

本实验使用 ORL 人脸库进行识别，结合 PCA 进行降维，再使用 KNN 完成分类。

| 项目 | 取值 |
| --- | --- |
| 数据集 | ORL 人脸库 |
| 分类个数 | 40 类 |
| 每类图片数 | 10 张 |
| 总图片数 | 400 张 |
| 图像尺寸 | 92 × 112 |
| 单张图像原始特征维度 | 10304 维 |
| 任务 | 40 类人脸识别 |

当前代码默认使用全部 40 个类别参与实验，不额外筛选子集。

## 2. 默认实验参数

默认参数定义在 `lab4.py` 的 `CONFIG` 中，当前设置如下：

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `num_classes` | 40 | 参与分类的人脸类别数 |
| `train_ratio` | 0.7 | 训练集比例 |
| `pca_components` | 50 | PCA 降维后的特征维度 |
| `knn_k` | 1 | KNN 邻居数 |
| `knn_metric` | `cosine` | KNN 距离度量 |
| `random_seed` | 42 | 训练/测试划分随机种子 |

在 `train_ratio = 0.7` 的默认设置下：

| 数据划分 | 数量 | 说明 |
| --- | --- | --- |
| 训练集 | 280 张 | 平均每类约 7 张 |
| 测试集 | 120 张 | 平均每类约 3 张 |

## 3. 参数扫描范围

为了分析不同参数对识别准确率的影响，程序额外做了如下扫描：

| 参数 | 扫描范围 |
| --- | --- |
| PCA 降维维度 | 5 到 100，步长 5 |
| KNN 的 K 值 | 1 到 15 |
| 训练集比例 | 0.3 到 0.9 |
| 分类个数 | 1 到 40 |
| 距离度量 | `euclidean`、`manhattan`、`cosine`、`chebyshev` |


## 4. 各参数对准确率的影响

### 4.1 PCA 降维维度

![PCA维度与准确率](./pca_dimension_accuracy.png)

从结果图可以看到：

- PCA 维度过低时，信息损失明显，5 维时准确率最低，只有约 82.5%。
- 当维度提升到 10 维以上后，准确率明显上升，并长期稳定在较高水平。
- 综合当前实验结果，40 到 70 维是相对稳定的有效区间，准确率大多在 93% 以上。
- 维度继续增大到 80 维以后，准确率没有继续提升，反而有一定波动，说明保留过多维度可能会把噪声一并引入分类。

结论：PCA 维度不是越大越好，选取中等维度通常能在信息保留和噪声抑制之间取得较好平衡。本实验中 `50` 维是一个合理选择。

### 4.2 KNN 的 K 值

![K值与准确率](./knn_k_accuracy.png)

从结果图可以看到：

- `K = 1` 时准确率最高，是当前实验的最佳选择。
- 随着 K 值增大，准确率整体呈下降趋势。

结论：对于 ORL 这种每类样本较少、类间差异较明显的数据，较小的 K 更适合，`K = 1` 或 `K = 3` 通常优于更大的 K。

### 4.3 训练集比例

![训练集比例与准确率](./train_ratio_accuracy.png)

从结果图可以看到：

- 随着训练集比例从 0.3 增加到 0.8，准确率总体逐步提升。
- 训练样本越多，模型能够学习到的类内变化越充分，因此识别效果更稳定。
- 当训练集比例继续提高到 0.9 时，准确率略有回落，说明在小样本实验中，测试集过小也会导致评估结果波动。

结论：一般来说，增加训练样本通常有利于提高准确率。但对于小样本，过小的测试集会导致评估结果受划分影响波动明显。本实验采取7:3的划分。

### 4.4 距离度量方式

![距离度量与准确率](./knn_metric_accuracy.png)

从当前结果图可以看到：

- `cosine` 距离的准确率最高。
- `euclidean` 和 `manhattan` 的表现接近，也能取得较好的效果。
- `chebyshev` 距离效果最差，明显低于其他三种度量方式。

结论：在人脸特征经过 PCA 处理后，`cosine` 距离更适合衡量样本方向上的相似性，因此识别效果最好。

## 5. 综合结论

结合当前实验结果，可以得到如下经验结论：

- 数据集规模为 40 类、400 张图像，默认划分下训练集 280 张、测试集 120 张。
- PCA 维度建议取中等大小，本实验中 40 到 70 维较稳定，默认 50 维合理。
- KNN 中较小的 K 更适合该任务，`K = 1` 效果最好。
- 训练集比例增大通常会提升准确率，`0.7` 到 `0.8` 的效果较好。
- 在当前实验结果中，`cosine` 距离优于其他距离度量。

因此，本实验较优的一组参数可以总结为：

| 参数 | 推荐值 |
| --- | --- |
| 分类个数 | 40 |
| 训练集比例 | 0.7 到 0.8 |
| PCA 维度 | 50 左右 |
| KNN 的 K 值 | 1 |
| 距离度量 | `cosine` |
| 准确率 | 0.9583 (95.83%) |
